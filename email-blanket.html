An Overview Of Email Components

Common email components used in fundraising, marketing, newsletter, update emails include:

  • Images
    • In-wrapper, full-width images
    • Nested images in other components
    • Full-bleed images
  • Callout boxes
    • Text only
    • Highlighted banner heading
    • With button
  • Buttons
    • Regular rectangular buttons
    • Completely rounded, pill-shaped buttons
    • Rounded corner buttons
  • Columns
    • Standard: Left-to-right / Top-to-bottom
    • Reversed: Left-to-right / Bottom-to-top
    • Rounded corner buttons
  • Footers
    • Social icons
    • Standard info: Address, Number, Organization name
